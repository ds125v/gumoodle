The two language folders that have been tested is en (for moodle 1.5.x) and en_utf8 (for moodle 1.6.x or above).

The sk, ja_utf8, ja, es, de_utf8, and fr folders are included here copied from the original quickmail package.
There might be some missing strings when you apply these language packs. If you have any contribution to these
language packs and would like them to be included in the next version please contact with Wen Hao Chuang at
wchuang@sfsu.edu
