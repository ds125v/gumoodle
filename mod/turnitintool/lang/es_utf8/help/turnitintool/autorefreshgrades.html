Ejercicios Turnitin - Actualizar las notas/puntuación automáticamente

Por defecto, el plugin de integración de Moodle buscará automáticamente sincronizar los cambios efectuados después de cada llamada a Turnitin. Usted puede usar esta opción para apagar esta función; sin embargo, para mantener la sincronización entre estos dos sistemas, usted debe refrescar manualmente y con frecuencia las notas y puntuación de originalidad dentro de Moodle.
