Trabalhos Turnitin - Tamanho Máximo do Arquivo

Essa configuração determina o tamanho máximo do arquivo para os envios dos usuários para cada parte do trabalho. O valor máximo que você pode configurar para esse valor é ditado pelo valor definido nas configurações do curso, esse valor ainda é limitado a um tamanho máximo de 20Mb que é o tamanho máximo permitido para upload de arquivo para o Turnitin.
