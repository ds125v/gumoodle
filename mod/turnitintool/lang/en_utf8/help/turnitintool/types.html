Turnitin Assignments - Submission Types

Submissions can be made in three different formats.

  1. File Upload

    Allows the user to upload files of the following type:

    • Microsoft Word (doc,docx,rtf,html)
    • PDF (pdf)
    • Plain Text (txt)

  2. Text Submission

    Allows the user to copy and paste plain text into a text area for upload to Turnitin.

  3. Web Page

    Allows the user to add the URL of a single web page. A snapshot of this web page is captured and uploaded to Turnitin as HTML.
