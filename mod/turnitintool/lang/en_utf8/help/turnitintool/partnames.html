Turnitin Assignments - Part Names

Determines whether the Parts of this assignment are Named Parts or this assignment is a Portfolio assignment.

  • Named Parts

    This setting allows you to set a part name and number of marks allocated to each assignment part. It also allows you to set individual start date, due date and post date for each assignment part.

  • Portfolio

    This setting allows you to group the assignment parts into one assignment. Each part is automatically assigned the overall default assignment start date, due date and post date. The parts are not named and users submit a set number of submissions. The overall grade for the assignment is calculated using an average of the grades given for each submission.
