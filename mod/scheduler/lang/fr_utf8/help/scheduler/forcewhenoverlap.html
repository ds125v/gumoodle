Forcage de l'ajout de crénaux

Ce controle permet de forcer la commande d'ajout de nouveaux créneaux même si les créneaux ajoutés entrent en conflit avec d'autres créneaux. Dans ce cas, seuls les créneaux qui peuvent être ajoutés le seront, les autres seront ignorés.

Si cette option n'est pas activée, l'ajout de créneaux vous proposera de supprimer les créneaux avant de pouvoir réaliser définitivement l'insertion.
