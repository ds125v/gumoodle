JW FLV Player Logo & Logo link

Logo: A logo image which appears at the bottom left of the video player window. Can be selected from the Moodle course files directory or copy and paste a URL link.

Logo link: A web page or file in the course files directory that users are directed to when they click on the logo. Can be selected from the Moodle course files directory or copy and paste a URL link.

Please note: Logo & logo link only function in licensed versions of the JW Player
