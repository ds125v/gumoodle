FLV Player Activity Module

Activity Module plugin for Moodle by Matt Bury
matbury@gmail.com
http://matbury.com/

JW FLV Player by Jeroen Wijering
http://www.longtailvideo.com/

This activity module leverages the complete range of functions in Jeroen Wijering's FLV Player. The mediaplayer.zip file is only included as the complete download package from Jeroen Wijering's website: http://www.longtailvideo.com/ It also contains some sample media, if needed, for testing the module. For all enquiries regarding the media player and its functions, please refer to Longtail Video.

To install:
* Upload the complete 'flv' (not 'FLV') directory to ***MOODLEROOT***/mod/
* Login to Moodle as administrator and in the 'Site Administration' panel, select 'Notifications'
* Moodle will install the FLV Player and DB table for you.

To use:
* Go to the course page and position you wish to deploy an FLV Player activity.
* Select 'FLV Player' from the 'Add an activity...' list.
* Enter the required parameters.
* Select or upload an FLV video file. In some cases, depending on your Moodle server setup, large video files must be uploaded via FTP.
* All other parameters are on default settings. Change or leave them as you wish.
* Select 'Save changes' (Moodle 1.8) or 'Save changes and preview' (Moodle 1.9).
That's it!

Google Code page: http://code.google.com/p/moodle-flv-player/
Moodle Docs page: http://docs.moodle.org/en/FLV_Player

X-moov provides media pseudo-streaming and has been included but not implemented yet.
